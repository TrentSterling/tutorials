# Game Development Tutorials

This repository contains a collection of deep-dive tutorials on essential game development algorithms and systems.

## Roadmap

- [x] **BSP (Binary Space Partitioning)**: Efficiently partitioning 3D space for visibility and collision.
- [x] **Spatial Hash**: Fast broad-phase collision detection using grid-based hashing.
- [x] **SAT (Separating Axis Theorem)**: The industry standard for convex shape collision detection.
- [x] **Radix Sort**: Non-comparative integer sorting for GPU-friendly particle systems and depth sorting.
- [x] **A***: The foundational pathfinding algorithm for grid-based and graph-based movement.
- [x] **Threading**: Managing concurrency, worker pools, and data synchronization in game engines.
- [x] **Navmesh**: Generating and traversing navigation meshes for complex AI movement.
- [x] **Platformer Pathfinding**: specialized pathfinding for 2D/3D platformers involving jumps, falls, and ledge grabs.
